# Memory

На уровне дизайна Rust создан так, чтобы избежать ошибок работы с памятью, которые свойственны таким языкам как C или C++.

## Memory issue C

В качестве примера приведу ошибку, с которой легко столкнуться в C: С не ограничивает работу с памятью, поэтому при неаккуратном использовании мы можем случайно взаимодействовать с той памятью, которую мы не должны были трогать. Это открывает возможности поиска уязвимостей и их использования.

#include

int main(void)
{
    char *s = "HI!";
    printf("%c\n", *s);
    printf("%c\n", *(s+1));
    printf("%c\n", *(s+2));
    printf("%c\n", *(s+50000));
}

В пятой строке этого кода мы пытаемся обратиться к той памяти, которая находится далеко за пределами нашего кода, то есть это баг.
